# Scribble Assignment Constitution

## Principles

1. Extend existing code. Do not rewrite.
2. Spec before implementation.
3. Every feature must have acceptance criteria.
4. Implement incrementally.
5. Validate each feature before moving on.

## AI Usage

1. AI suggestions must be reviewed.
2. Do not blindly accept generated code.
3. Specs are source of truth.

## Review Discipline

1. Match implementation to spec.
2. Test manually in two browser tabs.
3. Commit frequently with meaningful messages.
