## State Transitions

```
Room created  →  status: "lobby"
                 hostId: creator's participantId
                 participants: [creator]

Player joins  →  status: "lobby" (unchanged)
                 participants: [..., joiner]

Host starts   →  status: "game"
(POST /rooms/:code/start, host only, ≥2 players)

Clients poll  →  detect status === "game"  →  navigate to /game
```

## Validation Rules

| Field | Rule | Layer |
|-------|------|-------|
| `playerName` | trim, min length 1 | client + server (Zod) |
| `roomCode` (join form) | trim, min length 1 | client only (before API call) |
| `participantId` (start) | must match `room.hostId` | server |
| participant count (start) | `>= 2` | server |
