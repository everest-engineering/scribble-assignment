# Quickstart

## Running the Application Locally

1. Start the Backend:
   ```bash
   cd backend
   npm run dev
   ```

2. Start the Frontend:
   ```bash
   cd frontend
   npm run dev
   ```

3. Testing the Flow:
   - Open two browser windows (e.g., normal and incognito).
   - In window 1, create a room (you are the Host).
   - In window 2, join the room using the code.
   - In window 1, click "Start Game".
   - Verify that one of the windows becomes the "Drawer" and sees 3 word options, while the other window becomes a "Guesser" and sees a waiting message.
