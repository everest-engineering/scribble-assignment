# Quickstart: Game Start and Drawer Flow

## Manual Validation

1. Start backend and frontend.
2. Create a room as host.
3. Join from a second browser tab.
4. Start the game as host.
5. Confirm the host or first player is clearly shown as drawer.
6. Confirm the drawer sees the secret word.
7. Confirm the guesser sees that the secret word is hidden.
8. Try starting as non-host and confirm it is rejected.
9. Try starting with only one player and confirm it is rejected.
