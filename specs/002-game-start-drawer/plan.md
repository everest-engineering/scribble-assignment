# Implementation Plan: Game Start and Drawer Flow

**Branch**: `feat/scribble-spec-kit` | **Date**: 2026-05-29 | **Spec**: [spec.md](./spec.md)  
**Input**: Feature specification from `specs/002-game-start-drawer/spec.md`

## Summary

When the host starts the game, the room transitions to `active` and the host becomes the drawer. A deterministic word-selection function (char-code sum of room code modulo word list length) picks the secret word — no randomness. The backend viewer-scopes its snapshot response: the drawer receives the actual word; guessers receive a character-count placeholder. The game screen polls every ~2 seconds and displays the drawer's identity and the viewer-appropriate word.

## Phase 0: Research

All unknowns resolved. See [research.md](./research.md).

Key decisions:
- `selectWord(code, words)` = `words[charCodeSum(code) % words.length]` — pure, exported
- Viewer-scoped via `toRoomSnapshot(room, viewerParticipantId)` — existing signature
- `RoomSnapshot` gets two optional fields: `secretWord?` (drawer only) and `wordPlaceholder?` (guesser only)
- `drawerId: string` always present (empty string `""` in lobby)
- Placeholder format: one `_` per char, space-separated (e.g., `"_ _ _ _ _"` for "apple")

## Phase 1: Design & Contracts

### Backend Changes

#### 1. `backend/src/models/game.ts`

Add two fields to `Room`:
```typescript
drawerId: string;   // "" until game starts; participant.id of drawer when active
secretWord: string; // "" until game starts; the selected word when active
```

Add two optional fields + `drawerId` to `RoomSnapshot`:
```typescript
drawerId: string;          // always present; "" in lobby
secretWord?: string;       // drawer-scoped only
wordPlaceholder?: string;  // guesser-scoped only
```

#### 2. `backend/src/services/roomStore.ts`

New exported function:
```typescript
export function selectWord(code: string, words: readonly string[]): string {
  const index = [...code].reduce((sum, ch) => sum + ch.charCodeAt(0), 0) % words.length;
  return words[index];
}
```

Update `createRoom` to initialize new fields:
```typescript
drawerId: "",
secretWord: "",
```

Update `startRoom` to set them on transition:
```typescript
room.drawerId = room.hostId;
room.secretWord = selectWord(room.code, STARTER_WORDS);
```

Update `toRoomSnapshot` to scope the word:
```typescript
export function toRoomSnapshot(room: Room, viewerParticipantId?: string): RoomSnapshot {
  const isDrawer = room.status === "active" && viewerParticipantId === room.drawerId;
  const isActive = room.status === "active";

  return {
    code: room.code,
    hostId: room.hostId,
    status: room.status,
    participants: room.participants.map((p) => ({ ...p })),
    availableWords: isActive && !isDrawer ? [] : listWords(),
    roles: [...STARTER_ROLES],
    drawerId: room.drawerId,
    ...(isActive && isDrawer ? { secretWord: room.secretWord } : {}),
    ...(isActive && !isDrawer ? { wordPlaceholder: [...room.secretWord].map(() => "_").join(" ") } : {}),
  };
}
```

No changes required — `GET /:code` already passes `participantId` query param to `toRoomSnapshot`.
