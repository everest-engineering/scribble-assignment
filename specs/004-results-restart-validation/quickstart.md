# Quickstart: Results, Restart, and Final Validation

## Manual Validation

1. Start a two-player game.
2. Submit the correct word from a guesser.
3. Confirm all players see results after polling.
4. Confirm results show correct word, final scores, winner, and full guess history.
5. Attempt restart as non-host and confirm it is blocked.
6. Restart as host.
7. Confirm all players return to lobby.
8. Confirm the same players and room code remain.
9. Confirm drawer, word, drawing, guesses, scores, and result state are cleared.
